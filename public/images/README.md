# Images

Put your images here. Required:

- `hero.webp` — main hero image, ~1200x1200 or similar, **max 100 KB**.
  Convert your PNG/JPG to WebP at https://squoosh.app (free, no upload).
- `og-image.png` — social preview, exactly **1200x630**, max 200 KB.
  Design in Canva using your brand colors, export as PNG.
- Optional: `apple-touch-icon.png` — 180x180 PNG for iOS bookmarks.

Reference these paths in `src/config.ts`:
- `hero.heroImage: '/images/hero.webp'`
- `seo.ogImage: '/images/og-image.png'`
